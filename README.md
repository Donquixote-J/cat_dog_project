# 🐱🐶 Cat vs Dog Classification (纯手写 ResNet18)

这是一个从 0 开始搭建的深度学习计算机视觉实战项目。本项目基于 PyTorch 框架，**完全不使用任何预训练权重（From Scratch）**，构建并训练了一个 ResNet18 卷积神经网络，用于实现高精度的猫狗图像分类。

## ✨ 项目亮点

* **纯净复现**：从零手写 ResNet18 网络结构（`model.py`）及自定义残差块（Residual Block）。
* **鲁棒的数据加载**：自定义 `Dataset` 类（`dataset.py`），内置异常图片捕获与容错替换机制，支持动态数据增强（水平翻转、正则化）。
* **极速训练**：在单张 RTX 4090 GPU 上，仅耗时不到 3 分钟（10 Epochs）即可完成 2 万多张图片的训练闭环。
* **高准确率**：在未借助预训练模型的情况下，测试集准确率达到 **93.32%**，单张图片推理置信度极高。
